Why Notebook?

The Gaussian Process models require Python's scikit-learn library to make predictions. For accurate results, run the prediction code in your Cu3VS4_SelfValidating_BO.ipynb notebook.

How It Works

This system uses Gaussian Process regression with error correction to predict nanoparticle properties. Unlike traditional Bayesian optimization, it learns from prediction errors to improve future estimates.
